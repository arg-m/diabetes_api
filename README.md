# Diabetes Risk API

REST API для расчета риска диабета.  
Позволяет:

- Добавлять данные пользователя и рассчитывать риск (`POST /risk`)
- Получать данные конкретного риска (`GET /risk/{id}`)
- Получать список всех рисков (`GET /risks`)

> ⚠️ В текущей версии **ML модели нет**, используется простая формула для расчета риска:
>
> `risk = 0.3 * BMI + 0.5 * glucose + 0.2 * age`  

---

## Стек технологий

- Python 3.11  
- FastAPI  
- PostgreSQL  
- SQLAlchemy (ORM)  
- Pydantic  
- Uvicorn
